# 🏠 האפליקציה של לאוניד

אפליקציית נגישות עבור לאוניד - נכה צה"ל 100%+, דובר רוסית-עברית.

## ✨ מה כולל:
- 📅 **סידור עבודה חודשי** - עברית + רוסית (שמות: מטי, נתנאל, גומן, משה, סטס)
- 🏀 **כדורסל בני הרצליה** - אוטומטי מ-bhbasket.co.il + fallback ידני
- 💊 **טיפולים שבועיים** - בית הלוחם
- 🔊 **הקראה קולית** - עברית + רוסית

## 🏗️ מבנה הפרויקט:
```
leonid-app/
├── index.html                        ← הקובץ הראשי
├── netlify.toml                      ← הגדרות Netlify
├── netlify/functions/
│   └── get-basketball.js             ← פונקציה אוטומטית לכדורסל
└── data/
    └── (קבצי נתונים עתידיים)
```

## 📋 עדכון סידור עבודה חודשי:
1. פתח את `index.html` ב-GitHub
2. לחץ על ✏️ (Edit)
3. חפש את `workSchedule`
4. עדכן את התאריכים והשמות
5. לחץ "Commit changes"
6. Netlify מתעדכן אוטומטית!

## 🏀 כדורסל:
הנתונים נשלפים אוטומטית מ-bhbasket.co.il דרך Netlify Function.
אם הפונקציה לא עובדת - יש fallback עם נתונים ידניים.
